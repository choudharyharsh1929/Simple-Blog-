# Simple Blog Website

A simple and interactive blog website built using **HTML, CSS, and JavaScript**.  
Users can create, view, edit, and delete blog posts with categories/tags.  
The project uses a clean white interface with teal highlights and smooth animations.


## Features

- Add blog posts (title, content, tags)
- View all posts on the homepage
- Horizontal scrolling blog feed
- Edit and delete posts
- Animated and responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone the repository
2. Open `index.html` in a web browser
